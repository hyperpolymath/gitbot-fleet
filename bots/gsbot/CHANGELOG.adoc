=== Added
- Initial project structure and setup
- Discord bot implementation with comprehensive command framework
- Data models for Materials, Garments, Brands, and Users
- SQLite database with SQLAlchemy ORM
- Sustainability scoring system and algorithms
- Material impact analysis with environmental metrics
- Garment sustainability scoring
- Brand database with ratings and certifications
- User tracking system with gamification (points, levels, ranks)
- Core bot commands:
  - `!sustainability` - Get garment sustainability scores
  - `!impact` - View material environmental impact
  - `!alternatives` - Find sustainable alternatives
  - `!care` - Get garment care instructions
  - `!brands` - Search sustainable brands
  - `!compare` - Compare materials
  - `!tips` - Get sustainability tips
  - `!search` - Search garments and materials
  - `!profile` - View user profile
  - `!leaderboard` - View top users
  - `!setpreference` - Set user preferences
- Admin commands for data management
- Comprehensive test suite with pytest
- CI/CD pipeline with GitHub Actions
- Code quality tools (Black, Flake8, MyPy, Pylint)
- Sample data fixtures with 10+ materials, 7+ garments, 6+ brands
- Detailed documentation (README, CONTRIBUTING, CLAUDE.md)
- Environment configuration with .env support
- Logging system with colored console output
- Makefile for common development tasks
